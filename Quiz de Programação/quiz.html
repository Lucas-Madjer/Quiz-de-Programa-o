O que é uma variável em programação?

  • Um tipo de dado
  • Um valor constante
  • Um nome que armazena um valor que pode mudar
  • Uma função

O que é um loop (laço) em programação?

  • Um comando condicional
  • Um bloco de código que se repete enquanto uma condição for verdadeira
  • Uma função
  • Um método de depuração

O que significa "debugging" em programação?

  • Escrever código
  • Executar um programa
  • Encontrar e corrigir erros no código
  • Compilar um programa

Qual das seguintes opções é uma instrução condicional?

  • while
  • if
  • for
  • def

Em qual das seguintes linguagens de programação usamos "console.log()" para exibir mensagens no console?

  • Python
  • JavaScript
  • C++
  • Java

O que é uma função em programação?

  • Um tipo de dado
  • Uma variável
  • Um bloco de código reutilizável que realiza uma tarefa específica
  • Uma estrutura de controle

Qual a finalidade do comando "return" em uma função?

  • Repetir o bloco de código da função
  • Encerrar a função e retornar um valor para onde a função foi chamada
  • Declarar uma variável
  • Executar a função
